# Task Tide
A simple task management system that allows users to input tasks and their importance levels, and displays a prioritized list of tasks.

## Usage
1. Run the program using `python -m src.task_tide`.
2. Choose an option from the menu:
	* Add task: Enter task name and importance level.
	* Prioritize tasks: Display a prioritized list of tasks.
	* Save tasks: Save tasks to a file.
	* Load tasks: Load tasks from a file.
	* Exit: Quit the program.
