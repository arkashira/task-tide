# Task Tide
A task management system that allows users to input tasks and their importance levels, and displays a prioritized list of tasks.

## Usage
1. Create a new task by instantiating the `Task` class and adding it to the `TaskTide` instance.
2. Get the prioritized list of tasks by calling the `get_prioritized_tasks` method on the `TaskTide` instance.
3. Save the tasks to a JSON file by calling the `save_to_json` method on the `TaskTide` instance.
4. Load the tasks from a JSON file by calling the `load_from_json` method on the `TaskTide` class.

## Testing
Run the tests using `pytest` to ensure the implementation is correct.
